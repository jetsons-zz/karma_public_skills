# 动漫剧本生成

为动漫项目生成完整剧本。

## 触发条件

- 用户说 `/anime-script`
- 用户说 "生成剧本"、"写剧本"

## 前置条件

项目已通过 `/anime` 初始化。

## 执行流程

### 1. 读取项目配置

从 `~/.claude/skills/anime/projects/{project-name}/project.md` 读取：
- 主题
- 风格
- 时长
- 情绪关键词

### 2. 生成剧本内容

根据配置生成：

#### 故事梗概
一段话描述起承转合，包含：
- 开场设定
- 冲突/转折
- 高潮
- 结局

#### 角色列表
每个角色包含：
- **名称**：角色名
- **外观**：年龄、体型、面部特征（足够详细以生成图像）
- **性格**：性格特点
- **服装**：详细服装描述（颜色、款式、配饰）

#### 分镜概述
- 总镜头数
- 节奏安排
- 场景分布

### 3. 输出格式

```markdown
## 故事梗概

{一段话描述完整故事}

## 角色列表

### {角色1名称}
- **外观**：{详细外观描述}
- **性格**：{性格特点}
- **服装**：{服装描述}

### {角色2名称}
- **外观**：{详细外观描述}
- **性格**：{性格特点}
- **服装**：{服装描述}

## 分镜概述

共{X}个镜头，{Y}秒时长
- 镜头1-2：{场景描述}
- 镜头3-4：{场景描述}
- ...
```

### 4. 用户确认

展示剧本给用户，询问：
- 是否满意
- 是否需要修改
- 确认后保存

### 5. 更新项目状态

将剧本内容写入 `project.md`，更新阶段为 `storyboard`。

## 示例输出

```markdown
## 故事梗概

25岁的上班族阿杰在办公室里展现出惊人的手速和节奏感，看似在疯狂加班，
实则是在电脑上进行一场毫无意义的"敲木鱼"节奏游戏。就在他获得最高成就时，
却发现老板一直站在身后观战，而老板的反应却出人意料。

## 角色列表

### 阿杰
- **外观**：25岁左右的男性上班族，顶着淡淡的黑眼圈，身材偏瘦
- **性格**：虽然看起来疲惫，但一旦进入状态就会展现出极其夸张的行动力
- **服装**：白色衬衫（微皱）、深色西裤、简单手表、黑皮鞋

### 老板
- **外观**：典型的美式动画风格老板，秃顶，挺着一个巨大的啤酒肚
- **性格**：平时神情严肃且具有压迫感
- **服装**：灰蓝色西装外套、深灰色西裤、黑皮鞋

## 分镜概述

共6个镜头，30秒时长
- 镜头1-2：建立场景，展现阿杰"疯狂工作"
- 镜头3-4：揭示真相，其实是在玩游戏
- 镜头5：发现老板
- 镜头6：反转结局
```

## 下一步

剧本确认后，执行 `/anime-storyboard` 生成详细分镜。
