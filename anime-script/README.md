# Anime Script Skill

动漫剧本生成 Skill，为 Claude Code 提供剧本创作能力。

## 功能

- 生成故事梗概（起承转合）
- 创建详细角色列表（外观、性格、服装）
- 规划分镜概述

## 使用方法

```bash
# 前置条件：先执行 /anime 创建项目
/anime "一个搞笑的办公室动漫"

# 生成剧本
/anime-script
```

## 输出内容

- 故事梗概
- 角色列表（名称、外观、性格、服装）
- 分镜概述

## 依赖

需要先安装 `anime` skill。
