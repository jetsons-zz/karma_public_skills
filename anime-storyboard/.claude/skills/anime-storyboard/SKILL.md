# 动漫分镜生成

为动漫项目生成详细分镜列表。

## 触发条件

- 用户说 `/anime-storyboard`
- 用户说 "生成分镜"、"分镜脚本"

## 前置条件

剧本已通过 `/anime-script` 生成。

## 执行流程

### 1. 读取剧本

从 `project.md` 读取：
- 故事梗概
- 角色列表
- 分镜概述
- 总时长

### 2. 生成分镜列表

为每个镜头生成：

| 字段 | 说明 |
|------|------|
| 镜号 | 1, 2, 3... |
| 场景 | 场景位置（如：办公室-工位） |
| 画面描述 | 详细的视觉描述 |
| 对白/音效 | 角色台词或环境音效 |
| 时长 | 该镜头持续时间（秒） |
| 镜头 | 镜头类型和运动 |

### 3. 镜头类型参考

| 类型 | 说明 |
|------|------|
| 特写 | close-up，面部或物体细节 |
| 中景 | medium shot，上半身或局部场景 |
| 全景 | wide shot，完整场景 |
| 远景 | establishing shot，环境建立 |

### 4. 镜头运动参考

| 运动 | 说明 |
|------|------|
| 固定 | static，镜头不动 |
| 推 | zoom in，镜头推近 |
| 拉 | zoom out，镜头拉远 |
| 摇 | pan，镜头水平移动 |
| 升降 | tilt，镜头垂直移动 |
| 跟 | tracking，跟随主体移动 |

### 5. 输出格式

```markdown
## 分镜列表

| 镜号 | 场景 | 画面描述 | 对白/音效 | 时长 | 镜头 |
|------|------|----------|-----------|------|------|
| 1 | {场景} | {画面描述} | {对白/音效} | {X}s | {镜头类型} |
| 2 | {场景} | {画面描述} | {对白/音效} | {X}s | {镜头类型} |
| ... | ... | ... | ... | ... | ... |

总时长：{总时长}秒
```

### 6. 用户确认

展示分镜列表，用户可以：
- 修改某个镜头
- 增加/删除镜头
- 调整时长分配
- 确认后保存

### 7. 更新项目状态

将分镜列表写入 `project.md`，更新阶段为 `character`。
