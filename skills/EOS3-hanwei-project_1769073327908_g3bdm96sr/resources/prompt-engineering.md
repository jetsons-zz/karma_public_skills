# 提示词工程指南

## 从JSON到提示词的转换策略

### 1. 结构化组织

将JSON各字段按优先级组织为提示词结构：

[主体描述] + [风格定义] + [构图说明] + [色彩方案] + [光影效果] + [氛围营造] + [质量参数]

### 2. 权重分配

根据创作重点调整各元素的权重：
- 核心主体：高权重
- 风格特征：中高权重
- 细节修饰：中低权重
- 技术参数：低权重（作为后缀）

### 3. 关键词提取

从中文描述中提炼关键词：
- 保留核心名词和形容词
- 删除冗余的连接词和介词
- 使用专业术语替代口语化表达

### 4. 跨平台适配

不同AI绘画工具的提示词格式差异：
- Midjourney：注重自然语言，支持--参数
- Stable Diffusion：逗号分隔，支持权重标记
- DALL-E：完整句子描述

### 5. 迭代优化

基于生成结果反向调整：
- 对比原图与生成图的差异
- 识别描述不足的维度
- 补充缺失的关键特征
