# 动漫风格指南

## 日式动漫 (Anime)

**关键词:** `japanese anime style, cel shading, vibrant colors, anime aesthetic`

**特点:**
- 大眼睛、精致面部
- 鲜艳的色彩
- 清晰的轮廓线
- 动态的动作表现

**示例提示词:**
```
japanese anime style, young female character with long blue hair,
big expressive eyes, school uniform, cherry blossom background,
soft lighting, cel shading, vibrant colors, high quality animation
```

---

## 吉卜力风格 (Ghibli)

**关键词:** `studio ghibli style, hand-drawn, pastoral atmosphere, watercolor`

**特点:**
- 柔和的水彩质感
- 田园诗般的氛围
- 细腻的自然描绘
- 温暖的色调

**示例提示词:**
```
studio ghibli style, young girl in a meadow,
hand-drawn animation, pastoral atmosphere,
soft watercolor textures, warm sunlight,
gentle breeze moving grass, peaceful mood
```

---

## 皮克斯 3D (Pixar)

**关键词:** `3D pixar style, smooth rendering, warm lighting, cartoon 3D`

**特点:**
- 圆润的 3D 造型
- 温暖的光照
- 丰富的面部表情
- 鲜明的色彩对比

**示例提示词:**
```
3D pixar style, cute robot character,
smooth rendering, warm lighting,
expressive eyes, metallic texture,
colorful futuristic city background
```

---

## 赛博朋克 (Cyberpunk)

**关键词:** `cyberpunk anime, neon lights, futuristic, dark atmosphere`

**特点:**
- 霓虹灯光
- 高科技低生活
- 雨夜城市场景
- 紫/青/粉色调

**示例提示词:**
```
cyberpunk anime style, hacker character in neon-lit alley,
holographic displays, rain-soaked streets,
purple and cyan color palette, futuristic city,
high tech gadgets, moody atmosphere
```

---

## 水墨中国风 (Chinese Ink)

**关键词:** `chinese ink wash painting style, traditional, elegant brushwork`

**特点:**
- 黑白灰为主
- 留白艺术
- 山水意境
- 飘逸的线条

**示例提示词:**
```
chinese ink wash painting style, traditional scholar,
flowing robes in the wind, misty mountains background,
elegant brushwork, zen atmosphere,
black and white with subtle color accents
```
