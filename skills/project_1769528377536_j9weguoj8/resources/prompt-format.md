# 提示词文件格式说明

## 支持的格式

### 1. 纯文本格式（prompts.txt）
每行一个提示词，按顺序生成视频。

示例：
一只可爱的猫咪在阳光下打哈欠
城市街道上人来人往的繁忙场景
夕阳下的海滩，海浪轻轻拍打沙滩

### 2. YAML格式（prompts.yaml）
支持更丰富的配置选项。

示例：
videos:
  - prompt: '一只可爱的猫咪在阳光下打哈欠'
    duration: 5
    resolution: '1920x1080'
  - prompt: '城市街道上人来人往的繁忙场景'
    duration: 8
    resolution: '1920x1080'
  - prompt: '夕阳下的海滩，海浪轻轻拍打沙滩'
    duration: 6
    resolution: '1920x1080'

### 3. JSON格式（prompts.json）
{
  "videos": [
    {
      "prompt": "一只可爱的猫咪在阳光下打哈欠",
      "duration": 5,
      "resolution": "1920x1080"
    }
  ]
}

## 提示词编写技巧
1. 描述要具体：包含主体、动作、环境、光线、风格
2. 避免过于复杂的场景：单个提示词聚焦一个主要内容
3. 保持连贯性：相邻视频片段主题相关，便于衔接
4. 控制时长：建议单个片段5-15秒
