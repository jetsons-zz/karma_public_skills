# 笑话大师

专业的幽默内容生成与笑话创作助手。能够根据用户需求生成各类笑话（冷笑话、脑筋急转弯、段子、双关语等），支持中英文笑话创作，可按主题、场景、受众定制幽默内容。当用户说'讲个笑话'、'来点幽默'、'逗我笑'、'分享段子'时自动激活。适用于活跃气氛、内容创作、演讲开场、社交互动等场景。

## 使用场景

当用户请求娱乐内容、需要活跃气氛、想要听笑话或幽默段子时触发

**触发关键词:** 讲个笑话, 来点幽默, 逗我笑, 分享段子, 冷笑话, 脑筋急转弯, 笑话大师, 搞笑内容, 幽默一下

**激活模式:**
- `.*笑话.*`
- `.*幽默.*`
- `.*段子.*`
- `.*搞笑.*`
- `.*joke.*`
- `.*humor.*`

## 使用说明

=== 笑话大师参考文档 ===

[功能说明]
笑话大师是一个专业的幽默内容生成工具，提供以下核心功能：
1. 多类型笑话生成：冷笑话、脑筋急转弯、双关语、情景喜剧式段子
2. 主题定制：职场、生活、科技、动物、食物等多种主题
3. 受众适配：儿童友好、成人幽默、职场适用等不同风格
4. 双语支持：中文和英文笑话创作
5. 场景应用：演讲开场、社交破冰、内容创作素材

[技术规格]
- 笑话类型：冷笑话、脑筋急转弯、谐音梗、反转笑话、观察幽默、自嘲式幽默
- 语言支持：简体中文、英文
- 输出格式：纯文本、带解释说明、系列笑话集
- 质量标准：创意性、可读性、适用性、文化适配

[最佳实践]
1. 了解受众：根据听众年龄、文化背景选择合适类型
2. 控制长度：口头笑话建议3-5句，书面段子可适当延长
3. 测试反应：首次使用新笑话时观察受众反馈
4. 避免敏感：不涉及种族、性别、宗教等敏感话题
5. 情境匹配：正式场合用轻松幽默，非正式场合可更放松

[注意事项]
- 所有笑话内容健康向上，不包含不当内容
- 部分谐音梗和文化梗可能需要背景知识
- 幽默感受因人而异，同一笑话效果可能不同
- 使用时注意场合和氛围是否适合开玩笑

## 示例

### 基础笑话请求

**场景:** 用户想要听一个随机笑话来放松心情

**用户输入:**
```
讲个笑话
```

**预期输出:**
```
生成一个适合当前情境的笑话，包含笑话内容和必要的背景说明。例如：'为什么程序员总是分不清万圣节和圣诞节？因为 Oct 31 = Dec 25（八进制31等于十进制25）'
```

### 主题定制笑话

**场景:** 用户需要特定主题的笑话用于技术社群分享

**用户输入:**
```
给我讲个关于程序员的笑话
```

**预期输出:**
```
生成与程序员主题相关的幽默内容，确保技术受众能理解。例如：'程序员的三大谎言：1.这个bug明天就能修好 2.这次一定写注释 3.这代码不用测试也没问题'
```

### 儿童友好笑话

**场景:** 家长需要适合儿童的娱乐内容

**用户输入:**
```
给小朋友讲个笑话
```

**预期输出:**
```
生成适合儿童的健康幽默内容，语言简单易懂。例如：'小明问：为什么北极熊不吃企鹅？妈妈说：因为北极熊住北极，企鹅住南极，他们见不到面呀！'
```

### 脑筋急转弯

**场景:** 聚会游戏或团队建设活动

**用户输入:**
```
来个脑筋急转弯
```

**预期输出:**
```
提供互动式脑筋急转弯，先给问题让用户思考，然后揭晓答案。例如：'问题：什么门永远关不上？答案：球门！'
```

### 演讲开场笑话

**场景:** 专业演讲或培训开场

**用户输入:**
```
我需要一个适合演讲开场的笑话，主题是时间管理
```

**预期输出:**
```
生成与演讲主题相关的专业幽默内容，既能活跃气氛又能引入主题。例如：'大家知道为什么拖延症患者总说明天再做吗？因为他们认为明天会有48小时。今天我们就来聊聊如何让24小时发挥48小时的效果。'
```

## 常见问题

**问: 生成的笑话不好笑怎么办？**

答: 幽默感因人而异，可以尝试：1) 更换笑话类型（如从冷笑话换成脑筋急转弯）；2) 指定更具体的主题或场景；3) 要求生成多个笑话从中选择；4) 说明你的受众特征（年龄、文化背景等）以获得更适配的内容。

**问: 如何获取特定场合的笑话？**

答: 请明确说明使用场景，例如：'给我一个适合职场会议开场的笑话'、'需要儿童生日派对用的笑话'、'演讲中关于创新主题的幽默内容'。提供的信息越详细，生成的笑话越贴合需求。

**问: 笑话涉及敏感话题怎么办？**

答: 本技能严格遵守健康向上原则，不生成涉及种族、性别、宗教等敏感话题的内容。如果您认为某个笑话不当，请直接反馈，我们会立即调整。所有笑话均经过安全性审查。

**问: 能否生成英文笑话？**

答: 完全可以！您可以直接说'Give me an English joke'或'讲个英文笑话'。也支持中英双语对照模式，适合语言学习或跨文化交流场景。

**问: 如何保存喜欢的笑话？**

答: 您可以：1) 要求使用 Write 工具保存到文件，如'把这些笑话保存到 my_jokes.md'；2) 请求生成笑话合集并导出；3) 使用模板功能格式化输出后保存。系统支持 Markdown、JSON 等多种格式。

## 调试指南

=== 调试指南 ===

[常见问题排查步骤]

1. 检查输入清晰度
   - 确认是否明确说明了笑话类型或主题
   - 检查场景描述是否足够详细

2. 验证工具权限
   - 如使用 WebSearch 获取最新梗，确认网络访问权限
   - 如需保存笑话，确认 Write 工具已授权

3. 查看输出质量
   - 如果笑话不符合预期，尝试提供更多上下文
   - 可要求解释笑话的幽默点以理解创作逻辑

4. 测试不同类型
   - 依次尝试冷笑话、脑筋急转弯、谐音梗等类型
   - 找到最适合当前受众的笑话风格

5. 反馈与改进
   - 对生成的笑话给予反馈（好笑/不好笑）
   - 系统会根据反馈优化后续生成策略
